Q.How to create react app ?
Ans:
1) npm install -g create-react-app my-app -> this command will install the package and create react app.
2) npx create-react-app my-app -> Ye react app banyegi packeage ko udhar lekr.     

Q.How to strat react app ?
Ans: npm start -> This command start the react app.
